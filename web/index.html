Updating an existing Prism instance

  1. Edit the instance

  2. Go to the "Version" tab, click "Minecraft" in the list, click on "Change Version"

  3. Select the new version to use

  4. Go to the "Mods" tab and click on "Check for Updates". Confirm the following dialogs. Sometimes this crashes the whole launcher, in that case just try again :))
